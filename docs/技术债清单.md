### [P1-001] 单机模式房间协作功能不完整
- **描述**: 创建房间后无法真正进行协作，缺少任务分配和成员交互
- **影响**: 协作模式无法实际使用
- **发现日期**: 2026-05-08
- **发现人**: Agent
- **状态**: 🟢 已修复，房间列表、创建、详情页面、协作专区均已完善
- **修复内容**:
  - 房间创建后自动进入详情页面
  - 添加协作专区（任务发送、日志显示）
  - 添加解散房间功能
  - 添加成员自动刷新
  - 添加任务错误反馈机制
- **相关文件**:
  - `web_app.py`
  - `evolution/cluster/connection.py`
  - `web/static/index.html`

### [P1-002] 端口占用导致启动失败
- **描述**: 端口30000被占用时启动失败，没有自动切换端口
- **影响**: 用户无法启动服务
- **发现日期**: 2026-05-08
- **状态**: 🟢 已修复
- **修复内容**: `find_available_port` 函数已存在，但uvicorn调用方式有误，已修正
- **相关文件**:
  - `web_app.py`

### [P1-003] 命令行选择模型后Web页面不同步
- **描述**: 命令行选择的模型在Web页面不显示
- **影响**: 用户体验不一致
- **发现日期**: 2026-05-08
- **状态**: 🟢 已修复
- **修复内容**: init函数中添加了loadModelList()调用
- **相关文件**:
  - `web/static/index.html`
