# 玄枢接口清单

> 记录项目中所有REST API端点、参数、返回值及调用示例。

---

## 📍 API基础信息

- **Base URL**: `http://localhost:30000`
- **Content-Type**: `application/json`
- **认证方式**: 部分接口需要 `X-Approval-Token` 或 `X-Cluster-Token`

---

## 💬 对话相关

### POST `/api/chat`
处理对话消息

**请求参数**:
```json
{
  "message": "你好",
  "conversation_id": "可选，对话ID"
}
```

**响应**:
```json
{
  "success": true,
  "response": "回复内容",
  "conversation_id": "对话ID"
}
```

---

### GET `/api/conversations`
获取对话历史列表

**参数**:
| 参数 | 类型 | 必填 | 默认值 | 说明 |
|------|------|------|--------|------|
| limit | int | 否 | 20 | 返回数量 |

**响应**:
```json
{
  "success": true,
  "conversations": [
    {
      "conversation_id": "xxx",
      "title": "对话标题",
      "updated_at": "2026-05-08T10:00:00",
      "message_count": 10
    }
  ]
}
```

---

### GET `/api/conversation/{conversation_id}`
获取对话详情

**响应**:
```json
{
  "success": true,
  "conversation": {
    "conversation_id": "xxx",
    "title": "标题",
    "messages": [
      {"role": "user", "content": "用户消息"},
      {"role": "assistant", "content": "助手回复"}
    ]
  }
}
```

---

### DELETE `/api/conversation/{conversation_id}`
删除对话

**响应**:
```json
{"success": true}
```

## 🤖 模型相关

### GET `/api/models`
获取模型配置列表

**参数**:
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| force_reload | bool | 否 | 是否强制重新加载 |

**响应**:
```json
{
  "success": true,
  "models": [
    {
      "name": "配置名称",
      "provider": "ollama/openai_compatible/custom",
      "model_name": "实际模型名",
      "api_base": "http://localhost:11434",
      "api_key": "",
      "is_current": true,
      "has_api_key": false
    }
  ],
  "current_config": "当前配置名称"
}
```

---

### POST `/api/save_model`
保存模型配置

**参数** (FormData):
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| name | string | 是 | 配置名称 |
| original_name | string | 否 | 原始名称（编辑时用） |
| provider | string | 是 | 提供商类型 |
| model_name | string | 是 | 模型名称 |
| api_base | string | 是 | API地址 |
| api_key | string | 否 | API密钥 |

**响应**:
```json
{"success": true}
```

---

### POST `/api/switch_model`
切换当前模型

**请求参数**:
```json
{
  "name": "配置名称"
}
```

**响应**:
```json
{"success": true}
```

---

### GET `/api/ollama_models`


## 🏠 房间相关

### POST `/api/rooms/create`
创建房间

**请求参数**:
```json
{
  "room_name": "房间名称",
  "owner_name": "房主花名",
  "model": "使用的模型",
  "password": "可选密码"
}
```

**响应**:
```json
{
  "success": true,
  "room_id": "房间ID",
  "room_name": "房间名称"
}
```

---

### GET `/api/rooms/list`
获取房间列表

**响应**:
```json
{
  "success": true,
  "rooms": [
    {
      "room_id": "xxx",
      "room_name": "房间名",
      "owner_name": "房主",
      "owner_model": "模型",
      "members": [...],
      "total_members": 1,
      "has_password": false,
      "status": "active/waiting",
      "members_detail": [...]
    }
  ]
}
```

---

### GET `/api/rooms/current`
获取当前房间信息（需要Manager角色）

**响应**: 同 `/api/rooms/list` 中单个room对象，增加 `members_detail` 字段

---

### POST `/api/rooms/join`
加入房间（Worker）

**请求参数**:
```json
{
  "host": "192.168.1.100:30001",
  "port": 30001,
  "name": "成员花名",
  "mode": "auto/manual",
  "model": "使用的模型",
  "password": "房间密码（如果有）"
}
```

**响应**:
```json
{"success": true, "message": "已加入房间"}
```

---

### POST `/api/rooms/leave`
离开房间

**响应**:
```json
{"success": true}
```

---

### POST `/api/rooms/start_task`
启动协作任务（仅Manager）

**请求参数**:
```json
{
  "task_type": "code_generation",
  "description": "任务描述",
  "parameters": {}
}
```

**响应**:
```json
{
  "success": true,
  "task_id": "任务ID"
}
```

## 📊 其他接口

### GET `/api/info`
获取节点信息

**响应**:
```json
{
  "role": "standalone/manager/worker",
  "message": "Cluster disabled"
}
```

Manager模式:
```json
{
  "role": "manager",
  "room": {
    "room_id": "xxx",
    "room_name": "房间名"
  }
}
```

Worker模式:
```json
{
  "role": "worker",
  "node_id": "xxx",
  "connected": true,
  "mode": "auto",
  "model": "qwen2.5-coder:7b"
}
```

---
