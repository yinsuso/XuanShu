# 玄枢项目地图

## 📍 模块定位速查表

### 核心层

| 文件 | 定位 | 职责 | 关键方法 |
|------|------|------|----------|
| `agent.py` | 核心入口 | UniversalAgent主类 | `_process_simple()`, `run()` |
| `launcher.py` | 启动器 | 环境检测、模型选择、启动引导 | `run()`, `setup_ollama()` |
| `web_app.py` | Web服务 | FastAPI服务、REST API | 所有 `@app.` 端点 |
| `config.py` | 配置中心 | 全局配置常量 | 所有 `_*` 配置项 |
| `logger.py` | 日志系统 | 全局日志管理 | 日志格式化、文件输出 |
| `data_masker.py` | 数据脱敏 | 敏感信息掩码处理 | 数据脱敏函数 |
| `token_tracker.py` | Token追踪 | 模型调用Token统计 | 用量统计、按模型/日期聚合 |
| `evolution_engine.py` | 进化引擎 | 自进化能力入口 | 进化调度器 |

### 数据管理层

| 文件 | 定位 | 职责 | 关键方法 |
|------|------|------|----------|
| `model_providers.py` | 模型管理 | 配置、切换、调用模型 | `call_model()`, `config_manager` |
| `conversation_manager.py` | 对话管理 | 对话历史CRUD | `add_user_message()`, `save_current()` |
| `memory_core.py` | 核心记忆 | SQLite持久化记忆 | `save()`, `load()` |
| `memory_enhanced.py` | 增强记忆 | 向量搜索(可选) | - |

### 进化引擎层 (`evolution/`)

| 文件 | 定位 | 职责 |
|------|------|------|
| `multi_agent.py` | 多Agent协作 | 多个Agent协同工作 |
| `planner.py` | 任务规划 | 将复杂任务分解 |
| `react_loop.py` | React循环 | 推理+行动循环 |
| `reflection.py` | 反思机制 | 任务复盘与改进 |
| `role_switcher.py` | 角色切换 | 不同角色切换 |
| `skill_generator.py` | 技能生成 | 自动生成新技能 |
| `skill_validator.py` | 技能验证 | 验证技能有效性 |
| `trajectory.py` | 轨迹记录 | 执行轨迹存储 |
| `workflow.py` | 工作流定义 | 工作流DSL |
| `workflow_engine.py` | 工作流引擎 | 执行工作流 |

## 📦 跨文件调用速查

### Agent调用链
```
agent.py::UniversalAgent._process_simple()
    ├── model_providers.py::call_model()
    ├── skills_registry (动态加载)
    │   └── skills/code/python_exec/python_exec.py
    └── memory_core.py (如启用)
```

### Web请求处理链
```
web_app.py::api_chat()
    ├── conversation_manager.py::add_user_message()
    ├── agent.py::get_agent()._process_simple()
    └── conversation_manager.py::add_assistant_message()
```

### 房间创建链
```
web_app.py::create_room()
    └── ensure_cluster_initialized()
        ├── ClusterManager.create_room()
        ├── discovery.update_room_info()
        └── discovery.start_hosting()  # UDP广播启动
```

### 局域网房间发现链
```
web_app.py::/api/rooms/list
    └── manager.get_room_info()  (本地房间)
        discovery.get_available_rooms()  (UDP发现的远程房间)
            └── ClusterDiscovery._scan_loop()  # UDP 50005端口监听
```

### 集群节点加入链
```
web_app.py::join_room()
    └── ClusterClient.join()
        └── ClusterServer._handle_client()
            └── ClusterManager.add_node()
```

## 🎯 关键数据结构

### ModelConfig (model_providers.py)
```python
@dataclass
class ModelConfig:
    provider: ProviderType  # ollama/openai_compatible/custom
    name: str              # 配置名称
    model_name: str        # 模型名称
    api_base: str          # API地址
    api_key: str = ""      # API密钥
```

### ClusterNode (evolution/cluster/connection.py)
```python
class ClusterNode:
    node_id: str
    ip: str
    model: str
    role: str              # manager/worker
    mode: str              # auto/manual
    status: str            # online/offline
    capability_score: float # 能力评分 0.0-1.0
    connection: socket      # TCP连接
    pending_tasks: list    # 待处理任务
```

### Skill (skills/base.py)
```python
class Skill(ABC):
    name: str
    description: str
    parameters: list
    async def execute(**kwargs) -> dict
```

### ClusterDiscovery (evolution/cluster/discovery.py)
```python
class ClusterDiscovery:
    UDP_DISCOVERY_PORT: int = 50005
    broadcasting: bool       # 广播状态标志
    scanning: bool           # 扫描状态标志
    found_rooms: list        # 已发现的远程房间列表
    def start_hosting(self, extra_info: dict)
    def start_scanning(self)
    def update_room_info(self, room_name: str, room_id: str, extra_info: dict)
    def stop_hosting(self)
    def stop_scanning(self)
    def get_available_rooms(self)
```
