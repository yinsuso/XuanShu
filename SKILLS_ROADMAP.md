## 💡 技能开发指南

### 如何添加新技能

1. 在 `skills/` 目录创建新文件（如 `database_skills.py`）
2. 导入 `from skills.base import skill, SkillCategory`
3. 使用 `@skill` 装饰器定义技能
4. 实现技能函数

### 示例模板

```python
from skills.base import skill, SkillCategory
from skills.decorators import log_execution
from config import ALLOWED_DIR
from logger import logger


@skill(
    name="your_skill_name",
    description="技能描述",
    category=SkillCategory.CUSTOM,
    requires_confirmation=False
)
@log_execution
def your_skill_function(param1: str, param2: int = 0) -> str:
    """技能文档"""
    try:
        # 实现逻辑
        result = do_something(param1, param2)
        return f"✅ 成功: {result}"
    except Exception as e:
        return f"❌ 失败: {str(e)}"


logger.info("✅ 技能模块已加载")
```

### 技能命名规范

- 使用小写和下划线
- 动词开头：`read_`, `write_`, `create_`, `list_`, `find_`, `run_`
- 清晰表达功能
